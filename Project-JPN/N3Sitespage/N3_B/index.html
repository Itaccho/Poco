動詞[未然形]＋「ずに」　　名詞＋「をせずに」

不做A然后 no hacer A y.

  • 練習をせずにじょうずになれるはずがない。
  • わたしの親はわたしにいっさいゲームをさせずに、バイオリンの練習ばかりをさせられていました。

どんなに～ても　どんなに～でも

　「どんなに」＋イ形容詞 / 動詞[て形]＋「ても」　
　「どんな」＋名詞＋「でも」　
　「どんなに」＋ナ形容詞＋「でも」

  • どんなにがんばってもむくわれないことがある。
  • どんなにつらくても、前を向いて生きていかなければならない。
  • どんな人間でも失敗することはある。
  • どんなに元気でも、つねに体調管理を気にするべきである。

～ても　　～じゃなくても

不管是不是　　No importa si A

  • 今回の試験に落ちても、次回また受ければいいさ。
  • その役割はわたしじゃなくてもできると思います。なのでその仕事をわたしだけにたのむのはやめてください。
  • きょうは食欲が良いです。お昼、ごはんを食べてもお腹がいっぱいになりませんでした。
  • わたしでもなやみはひとつぐらいありますよ。

として

作为　　para + (persona)

  • 日本のレストランで冷たい飲み水が提供されることについて、わたしは中国人として、あまり良いとは思いません。
  • プロとしてのプライドを持つことと習慣を身につけることは重要だ。
  • この問題に関して、女性としての意見をお聞きしたいとおもいます。

～しにては　　～にしても

与A相比　　comparado con A

  • このワンピースは値段の安さにしては質がとても良い。
  • このレンジは値段の高さにしても質がとても良い。
  • 練習する時間が 1週間だけしか なかったにしては、失敗がひとつもなく、とてもじょうずに踊（おど）れていた。

動詞＋「としたら」

假设　　Si A ocurre

  • もし10年前に戻れるとしたらなにをやり直したいですか？
  • 未来のことが知れるとしたら、あなたはどうしますか？
  • もし1年間ディズニーランド入園無料チケットが手に入ったとしたら、毎日ディズニーランドに行きますか。
  • あした世界が終わるとしたら、最後に食べたい料理はなんですか？

動詞[て形]＋「たものだ」

以前很常常做　　Antes hacía A con frecuencia

  • ちいさいころは、よくおとうさんとサッカーボールで遊んでいたものだ。
  • ちいさいころはよくおばあちゃんにいろいろな場所に連れて行ってくれたものだ。
  • 2年前はよくともだちとごはんに行っていたものだ。

～のふりをする　　動詞＋「ふりをする」

～のように見せる、～のようにふるまう。
假装　　pretender
あまり良い意味では使われない

  • わかったふりをするな。
  • 見て見ぬふりをする
  • 知らんぷりをしても無駄だ。
  • 彼は自分に都合の悪い話は聞こえないふりをする癖がある。

～にとって

对（人）来说　　　　Para + (personas)

  • 彼女にとって山下くんはお金のなる木のような人だと思っているのではないか？
  • 鈴木先生はわたしにとって恩師の方です。
  • 現代人にとって、スマホは生活の一部である。

～のわりには

～とは思えないくらい　　　　Al contrario de

  • 部長はいつも忙しいと言っているわりに、仕事中によくスマホを見ている。
  • このラーメンは値段のわりには、おいしかった。
  • 60歳のわりには若く見えますね。

～のくせに

虽然说～可是　　　A pesar de que

  • あの占い師は占い師のくせに、彼自身の未来のことは当たらない。
  • あいつは自分でなにも努力していないくせに、頑張っているふりをしているのがうざい。
  • バイトして留学するって言っていたくせに、ぜんぜんバイトして貯金していないじゃん。

「～など / なんて」＋否定文

似た物事の中からもっとも代表的なものを上げる表現　他人や物事に対する軽蔑を表す　自分に対して使うと木は謙遜の意味にある。

Por ejemplo / descreciación

  • バク宙などできない。
  • 風邪なんかで病院に行く必要はないよ。
  • わたくしの作った作品なんて、ただ人から教わって作ったものですよ。
  • 君はスケジュール帳なんか使っても日程管理ができないだろ。
  • あいつなんかに負けたくない。

肯定文でも「～なんて」が使われる。肯定文のときは、上のみっつに加えて、驚きの意味を表す。

  • わたくしの作った作品なんて、ただ人から教わって作ったものですよ。
  • まさか彼が裏切り者だっただなんて。
  • おかあさんが難病をかかえていたんだなんて、今日初めて知りました。

名詞＋「のかわりに」　動詞＋「かわりに」

交换 代替　　　alternativo intercambio

  • 後輩のかわりに先輩が責任を負うこととなった。
  • コロナの時期は、外食するかわりにネット通販でレストランなどの食べ物を注文する。
  • 土曜日出勤したので、そのかわりに月曜日が休みとなった。
  • いまのアルバイトは給料がそこそこ良いかわりにやることがとても多い。

動詞[た形]＋「たせいで」　　名詞＋「のせいで」

因为 都怪　　　Por que / por (del sentimiento negativo)

  • 彼が邪魔をしたせいで 電車を乗り過ごしてしまった。
  • タバコのせいで、多くの人が健康を害している。
  • 甘いものがすきな せいで糖尿病になった。
