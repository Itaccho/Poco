N5语法的小测验

助詞「の」

check!

它的意思是「的」，但是它只被用再两个名词的中间。

例

  • わたし の せんせい
  • あした の あさごはん
  • かれ の パソコン
  • にほん の ぶんか

あの人（ひと）は だれですか？

check!

動詞（どうし）「ある」「いる」

check!

东西→「ある」　人・动物→「いる」

很常见的错误

  • わたしのともだちは とうきょうにあります
  • わたしの へやに いぬがあります
  • おかあさんがあります
check!

助动词「です」和「ます」怎么区别？

用「AはBです」的时候，A跟B是同一的关系。「ます」被用在一个动词的后面。
