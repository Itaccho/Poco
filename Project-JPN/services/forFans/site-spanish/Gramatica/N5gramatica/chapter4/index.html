Las partículas （じょし）

Las partículas en japonés se llaman じょし

  • los tipos de じょし
    • 「は」: nominativo
    • 「が」
    • 「も」
    • 「の」
    • 「に」
    • 「で」
    • 「と」
    • 「へ」
    • 「から」
    • 「まで」
    • 「を」: propósito
    • 「だけ」

    vvv　　¿Cómo se usan estas partículas? 　vvv

    Se usa「は」para contar el carácter de persona o la situación etc　(¿Cómo es?)

    しょうは　にほんごの　せんせい　です。

    わたしは メキシコに すんでいます。

    Se usa「が」 para contar　(¿Qué o quién es o hace algo?)

    わたしは　ゲームをすることが　すきです。

    さいとうさんではなく、きむらさんがミーティングにきます。

    このおみせでは、カードがつかえません。

    わたしは　わるくありません。　あなたがわるいです。

    このりょうりは、わたしの　おとうさんが　つくったものです。

    「も」significa "también"

    Se mete「の」en dos sustantivos

    わたしの　おかあさん

    にほんごの　せんせい

    「のもの」 posesiones

    このペンは　わたしのです。（このペンは　わたしのものです。）

    このスマホは　だれのですか？


    「の」　＝　「こと」 o 「ことば」

    わたしが　いったのを　かいてください。

    かれが　いったのは　「ありがとう」です。

    「の？」　preguntas informales

    ケンくんは　いま　なにを　しているの？

    「に」

    1. existencia:　「にある」「にいる」/「にあります」「にいます」
    2. destino:　「にいく」
    3. un objeto de acciones: 「でんしゃに乗のる」　「ここになまえを書かいてください。」　「食たべにいく」　「買かいに行いく」
      「かれは　コンビニへ　のみものを　かいに　いきました。」　（かう comprar　＋「に」）
      「おとうさんは　いぬと　さんぽを　しに　でかけました。」
    4. un sitio donde vives: 「にすんでいる」/「にすんでいます」

    Traduce estas oraciones.

    Mi padre va a Tokio.

    Mi hermano mayor vive en Hiroshima.

    Kei va al supermercado para comprar verduras.

    Junn salió para pasear al perro.

    Se usa「で」para decir donde

    「渋谷しぶや駅えきの出口でぐちで ともだちと あう」Quedar con amigos en la salida de la estación Shibuya.

    「家いえでごはんをたべる」 comer en casa

    Usamos じょし「と」para....

    1. añadir un más sustantivo 「あなたとわたし」
      2. すずきさんは いつも トマト と みかんを かいます。
    2. conectar un nombre de una persona CON quien hacer 「ともだちと会う」
    3. おねえちゃんに「ゲームをしよう」といいました。

    Se usa「へ」para decir un destino

    países / ciudades lejanas

    きょうとへ　かんこうをしに　いきたいです。

    「から」significa "desde"

    このページのさいしょから　よみます。

    「まで」significa "hasta"

    このしゅくだいを　らいしゅうまでに　して　ください。

    Ponemos「を」detrás de un sustantivo y ante de un verbo. En este caso, el verbo se llama 「たどうし」.

    わたしは　まいにち　ランニングを　します。

    おかあさんが　わたしに　でんわを　します。

    あさごはんを　たべますか？

    おんがくを　ききます。

    「だけ」significa "solamente" "solo"

    この家いえだけ古ふるい。

    そこだけ　よく　わかりません。

    las terminaciones de los vervos de japonés

    • う
    • く
    • す
    • つ
    • む
    • る
    • える

    La forma original de verbos + 「て」

    Cada terminación se convierte así.

    • う　　　--＞って
    • く　　　--＞いて
    • す　　　--＞して
    • つ　　　--＞って
    • む　　　--＞んで
    • る　　　--＞って
    • える　　--＞えて
    • 「かく」→「かいて」　　「あく」→「あいて」　　「ひらく」→「ひらいて」　　「うごく」→「うごいて」　　「いる」→　「いて」
    • 「はなす」→「はなして」　　「さがす」→「さがして」　　「する」→「して」　　「うんどうする」→「うんどうして」
    • 「おわる」→「おわって」　　「いく」→「いって」　　「おもう」→「おもって」　　「はいる」→「はいって」　　「のる」→「のって」　　「かう」→「かって」　　「うる」→「うって」　　「ある」→「あって」　　「たつ」→「たって」　　「かつ」→　「かって」
    • 「よむ」→「よんで」　　「すすむ」→「すすんで」　　「のむ」→「のんで」
    • 「しらべる」→「しらべて」 「おしえる」→「おしえて」 「いれる」→「いれて」 「でる」→「でて」

    La forma progresiva「～ている」

    1. 「～ている」= "estar" + V~ando ahorita

    「今いま」 ahora.

    • べんきょうする　→　べんきょうしている
    • いま　にほんごの　べんきょうを　しています
    • かわのそばで　ランニングを　しています
    • かく　→　かいている
    • おぼえる　→　おぼえている
    • ケンくんは　いま　なにを　しているの？
      いま　ほんを　よんでいるよ。

    2. una acción seguida mucho tiemo

    • わたしは　メキシコに　すんでいます。　（すむ）
    • あのひとを　しっています。　　（しる）
    • わたしがいったことを　おぼえていますか？
    • わたしがいったことを　おぼえてる？

    3. rutinas

    • まいしゅう　えをかいています
    • よく　おんがくをきいています

    4. situación / condición

    • さとうさんは　かぜを　ひいています。
    • ふくしまさんは　いま　ちこくしています。
    • けいたくん、げんきにしている？
      うん、げんきだよ。
    • きょう（わたし）は　すこし　つかれています。
    • かれは　たぶん　ねています。
    • かのじょは　いま　うんてんに　しゅうちゅう　しています。

    5. trabajo

    • カフェのてんいん　を　しています。
    • けいさつかん　を　しています。
    • キッチンのしごと　を　しています。
      コック　の　しごと　を　しています。
    • タクシーの　うんてんし　を　しています。
    • ホテルの　うけつけの　しごと　をしています。

    みらいの ことを はなす　hablar del futuro

    「みらい」significa "futuro"

    Vocabulario del tiempo
    • いちじかんご（一時間後）
    • あした（明日）
    • あさって
    • みっかご（三日後）
    • いっしゅうかんご（一週間後）
    • らいしゅう（来週）
    • らいげつ（来月）
    • はんとしご（半年後）
    • らいねん（来年）
    • しょうらい（将来）
    1. La forma original（げんけい）
    2. 「つもりです」
    3. 「よていです」es más formal en estas tres palabras.
      por ejemplo
    1. あした　わたしの　ともだちが　いえにきます。
    2. わたしは　 来らい週しゅうやすむつもりです。
    3. やまださんが3みっ日後かごこちらに　もどってくる よていです。

    Imperativo : 「めいれいけい」

    「めいれいけい」tiene tres formas

    • u音おん どうし:　「かく」→「かけ」
    • e音どうし:　「おぼえる」→「おぼえろ」
    • i音どうし:　「おきる」→「おきろ」
    • 「来くる」→「こい」

    「て」形けい

    • かいて
    • おぼえて
    • おきて
    • きて
      「ください」es la forma formal
    • かいてください
    • おぼえてください
    • おきてください
    • きてください

    「～したいです」

    「わたしは～したいです」significa "quiero"　Esa forma se llama「がんぼうけい」en japonés.

    「がんぼうけい」toma la forma「ます」形けい

    • わたしは　にほんへ　いきたいです。
    • わたしは　すしを　たべたいです。
    • わたしは　かいものを　したい。

    La conjugación de los verbos de japonés (3) verbo del pasado　「過去かこ形けい」

    Las palabras del pasado

    • いっぷんまえ　1分前
    • いちじかんまえ 1時間前
    • けさ　今朝　esta mañana
    • きのう　昨日　ayer
    • おととい　anteayer
    • みっかまえ　三日前　hace tres días
    • いっしゅうかんまえ　1週間前　una semana anterior
    • せんしゅう　先週　la semana pasada
    • いっかげつまえ　1カ月前　un mes anterior
    • せんげつ　先月　el mes pasado
    • きょねん　去年　el año pasado
    La forma del h5asado (かこけい) → verbo ＋「た」
    • 「かく」「かいた」　　「あるく」「あるいた」
    • 「たべる」「たべた」　　「やめる」「やめた」
    • 「のむ」「のんだ」　　「よむ」「よんだ」
    • 「する」「した」
    • 「くる」「きた」
