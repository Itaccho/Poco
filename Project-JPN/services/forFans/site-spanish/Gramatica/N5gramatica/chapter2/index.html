La autopresentación y las conversaciones básicas

はじめまして、わたしの　なまえは　○○です。よろしくおねがいします。

Las partículas 「は」「の」

Las partículas en japonés se llaman 「じょし」.

じょし「は」 es nominitivo

じょし「の」 se sustituye entre dos sustantivos.

わたし　の　おかあさん　mi madre
えみ　の　にほんご　の　せんせい　el profesor de japonés de Emi　（「にほんご」＝　el idioma japonés）

「わたしのもの」　"el mío"

その　ほんは　わたしのもの　です。 (sono honn wa watashinomono desu.)

El verbo「です」

El verbo「です」 funciona como afirmación. Eso viene detrás de sustantivos o adjetivos.

これ、それ、あれ

Estos son sustantivos que significa "este", "ese", "aquel".

これは　ペンです。　Esto es un boli.

この、その、あの ＋ un sustantivo

この、その、あの no son sustantivos, pero se funta con sustantivos.

このペン　este boli 　あのひとは　わたなべさん　です。　Aquella persona es Sr(Sra). Watanabe.
