parte 5

La forma negativa del pasado

la forma negativa + 「なかった」
u音（おん） → a e音 → e i音 → i　/ los otros
行く　→　いかない 止める　→　とめない 起きる　→　おきない
書く　→　かかない 開ける　→　あけない いる　→　いない
歌う　→　うたわない 考える　→　かんがえない 来くる　→　来こない

書かく　→　かかなかった

おぼえなかった

「～することができる」

Esa forma es igual como 「～すること」　「できる」significa "poder"

いえで歌うたを歌うことができる

「れる」「られる」
可能かのう形けい


かのうけい es la forma posible. Significa "poder"
「～すること」「できる」se corta a 「れる」o「られる」.

u音どうし:　母音ぼいんe(＝ el vocal "e") ＋る

  • 「いう」「いえる」
  • 「書かく」「書かける」
  • 「聞きく」「聞きける」

e音, 「来くる」:　られる

  • 覚おぼえる　⇒　覚おぼえられる
  • 開あける　⇒　開あけられる

「来くる」⇒「来こられる」

i音

  • 見みる　⇒　見みれる
  • 起おきる　⇒　起おきられる

じどうし　たどうし

自動詞じどうし　Verbos intransitivos

他動詞たどうし　Verbos transitivos

por ejemplo　他動詞:　「を書く」「を見る」「を覚える」「を集あつめる」「を開あける」　⇒　Necesita la partícula（じょし） 「を」

por ejemplo　自動詞:　「起きる」「開く」「行いく」

Sabes じょし?　Y ¿cómo los usamos?
