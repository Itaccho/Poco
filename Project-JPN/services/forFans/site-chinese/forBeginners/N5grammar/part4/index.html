N5単語と文法のテスト

日本語の要素

  • 主語（しゅご）
    　＜例＞「～は」「～が」
  • 述語（じゅつご）
    　＜例＞「する」「です」「ます」「だ」
  • 名詞（めいし）
    　＜例＞「わたし」「国（くに）」「家（いえ）」
  • 動詞（どうし）
  • 目的語（もくてきご）
    　＜例＞「でんわを」「しごとを」
  • 助詞（じょし）
    　＜例＞「は」「が」「に」
  • 助動詞（じょどうし）
    　＜例＞「ます」
  • 接続語（せつぞくご）
    　＜例＞「そして」「だから」「しかし」
  • 副詞（ふくし）
    　＜例＞「とても」「すこし」「ちょっと」
  • 数詞（すうし）
    　＜例＞「1個」「1つ」
  • 動詞の活用形（かつようけい）
    　＜例＞て形　ます形　否定形
  • 原形（げんけい）
  • 過去形（かこけい）
    　＜例＞「した」　「はなした」
  • 進行形（しんこうけい）
    　＜例＞「している」
  • 受身形（うけみけい）
    　＜例＞「された」
  • 送り仮名（おくりがな）
    　＜例＞「来る」の「る」

N5 part5

過去形

            N5 形容詞の過去形
            多い　　原型　現在形
            多かった　　過去形

ダメないいかた：　「多いかった」

把「多い」的「い」去掉，然后直接加「かった」就好。

動詞＋「た」

  • 買う　→ 買った　　　言う　→　言った
  • 話す　→　話した　　　消（け）す　→　消した
  • 受ける（うける）　　テストを受けた

    • 動詞＋「ました」　　「ました」＝「ます」＋過去「た」

  • けさ朝ごはんを食べました
  • 日本語を3か月間（かげつかん）勉強（べんきょう）しました。

  • 行く　（iku）　　行か ika　なかった
  • あそぶ(asobu)　　あそば asoba なかった
  • 食べる　taberu　　食べなかった
  • 来る　kuru　来(ko) ない　　来(ki) た　　来(ko) なかった

    • 過去形の否定形

    「しなかった」「しませんでした」

  • 行かなかった　行きませんでした
  • 食べなかった　食べませんでした
  • 遊ばなかった　遊びませんでした
  • 来（こ）なかった　　きませんでした

    • できる・られる

    動詞の原形＋「ことができる / ことができます」

    　　　行くことができます

　　　話すことができます

　　　書くことができます

　　　飲むことができます

    e形動詞＋「られる」　　u形動詞＋「る」

  • 覚えられる　覚えることができる
  • 書くことができる　書ける
  • 話すu　　話せeる　　　さしすせそ
  • 飲む　　飲める　　　　　まみむめも
  • 来ることができる　来（こ）れる
  • 見える

    「～したあと」 做A的后

  • 走（はし）ったあとはシャワーをあびる。
  • ごはんを食べたあとは歯をみがく。

    • 「～してから」 做完A之后开始

  • 仕事（しごと）が終わってから電話をします。

    • 形容詞＋「て」 然后

    イ形容詞＋「くて」

  • 外（そと）は暑くて あせがでる。
  • そのアニメは面白（おもしろ）くて、何時間（なんじかん）も見ました

  • ナ形容詞＋「で」

  • しずかさんは 元気（げんき）で、優（やさ）しいひとだ。
  • 日本語の文法は、かんたんではない / じゃない。
