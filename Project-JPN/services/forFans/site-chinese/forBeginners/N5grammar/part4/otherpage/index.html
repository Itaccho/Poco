N5 part4

文法の要素

  • 主語（しゅご）　＜例＞「～は」「～が」
  • 述語（じゅつご）　＜例＞「する」「です」「ます」「だ」
  • 名詞（めいし）　＜例＞「わたし」「国（くに）」「家（いえ）」
  • 動詞（どうし）
  • 目的語（もくてきご）　＜例＞「でんわを」「しごとを」
  • 助詞（じょし）　＜例＞「は」「が」「に」
  • 助動詞（じょどうし）　＜例＞「ます」
  • 接続語（せつぞくご）　＜例＞「そして」「だから」「しかし」
  • 副詞（ふくし）　＜例＞「とても」「すこし」「ちょっと」
  • 数詞（すうし）　＜例＞「1個」「1つ」
  • 動詞の活用形（かつようけい）　＜例＞て形　ます形　否定形
  • 原形（げんけい）
  • 過去形（かこけい）　＜例＞「した」　「はなした」
  • 進行形（しんこうけい）　＜例＞「している」
  • 受身形（うけみけい）　＜例＞「された」

这里是N5与N4的内容：怎么区别日语的助词？

の

「の」一定要把两个名词放在左右边


は　跟　が

「は」を使った例文：わたしは会社員です。わたしは～と思う　君は別（べつ）の色のかばんを買った。

用「は」的时候，此后面的情报比主语重要。

会社（かいしゃ）は東京（とうきょう）にあります。

「が」を使った例文：これがいい。　音楽をきくことが すきだ。

このケーキは、わたしが買ったものです。

「が」强调”谁“”什么东西“像这些的情报

A: 質問はありますか？

B: 質問はないですが、すこし 相談（そうだん）があります。


に

狭い範囲（はんい）の目的地（もくてきち）「わたしは アニメイトに 行きます。」「トイレに行く」

場所「その本屋（ほんや）にたくさんの本が売っている。」「ハンスは いま いえに います。」「犬（いむ）がテーブルの下に いる。」「砂糖（さとう）は その たな の 上に あります。」

　

時間、日（ひ）にち「きょうの夜9時に来てください。」「23日に会社（かいしゃ）の面接（めんせつ）があります。」

很窄的空间＋「に」＋动词　　「椅子（いす）にすわる」「さいふに おかねを入（い）れる」「机（つくえ）の上（うえ）に 本を置く（おく）」

目的の意味→動詞ます形＋「に」
アルバイトをしに行く　イヤホンを買いに行く


へ

京都へ旅行に行きます。　家へ帰ります。

で

手段（しゅだん）　「車（くるま）で会社（かいしゃ）に行く」　包丁（ほうちょう）で トマトを切（き）る

場所「東京で仕事をしたいです。」　「スーパーで食材を買う。」　「ここで ようふく を買います」

人数（にんずう）　「何名様（なんめいさま）ですか？ 3名（めい）です。」「ひとりで ここに 住（す）んでいます。」

程度（ていど）　「これでいいですか？」「こんな感じで良いですよ。」「18さいで ひとりぐらしを はじめます。」

あと20分で 東京（とうきょう）に 着（つ）きます。

と

お兄ちゃんとお姉ちゃんが けんかを し始めた。　みんなと一緒（いっしょ）にレストランでごはんを食べたい。

から～まで

朝から晩（ばん）まで働（はたら）く。

も

あなたも明日のレッスンにいきますか？

動詞の活用 　「～て」

  • u形動詞Ⅰ　話す　→　話して　　書く　→　書いて　　泣く　→　泣いて　　信じる　→　信じて　　出す　→　出して
  • u形動詞Ⅱ　行く　→　行って　　取る　→　取って　　使う　→　使って　　作る　→　作って　　言う　→　言って
  • e形動詞　越える　→　越えて　　得る　→　得て
  • i形動詞　見る　→　見て　　
  • 来る（くる）　→　来て（きて）

    • 現在進行形（げんざい しんこうけい）＝「～ている」

    「～ている」の形は、「～て」と同じ。

    未来の表現

    現在形と同じ形。よく使う単語は「あとで」「一時間後（いちじかんご）」「あした」「来週」「来月」「来年」

  • あした、またお会いしましょう。
  • 来年日本に留学（りゅうがく） します。
  • これからすこし街に出よう と思います。
  • 来月故郷に帰る つもり / 帰る 予定です。

    • 命令文（めいれいぶん）

    1. 動詞「て」形　
      　＜例＞ドアを開けて。
      　その意味を教えてください。
    2. e形動詞＋「ろ」　　覚えろ/閉めろ
    3. u形動詞＋eの母音　
      　聞け/立て/乗れ/書け/とまれ

    命令文（めいれいぶん）＋「ください。」

    動詞「て」形＋「ください。」　＜例＞教えてください
    　もういちど言ってください
    　そのメモを送ってください

    很常见的错误　よく見る間違い例

    例を見る +

    助詞　の

    • 不要把「の」放在形容词的后面***人が多いの 場所（ばしょ）
    • ナ形容詞に「の」***昨日有名（ゆうめい）の お店に行きました
    • 不要把「の」放在动词的后面***日本に住むのことがしたいです。

    助詞　は

    • 質問が ありませんが、相談があります。
    • わたしはわんちゃん（＝　いぬ）はほしいです。
    • ミキがわたしの彼女（かのじょ）です。

    在下面的句话都不能用へ(e)。

    • トイレへ行きます。
    • スターバックスへ行きます。
    • 自宅（じたく）へ帰（かえ）ります

    u形動詞Ⅰ

    • 出して　 出すして
    • 話して　 話せて　話すことをして

    u形動詞Ⅱ

    • 使（つか）って　使つて
    • 言って言つて
    • 立（た）って立て
