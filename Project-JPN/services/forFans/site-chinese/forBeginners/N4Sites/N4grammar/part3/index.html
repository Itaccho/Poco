  • 助詞（じょし）　＜例＞「は」「が」「に」
  • 助動詞（じょどうし）　＜例＞「ます」
  • 接続語（せつぞくご）　＜例＞「そして」「だから」「しかし」
  • 副詞（ふくし）　＜例＞「とても」「すこし」「ちょっと」

N4语法 part3

形容詞＋「そう」

看上去 好像

このケーキ、おいしそう。

この男は強そうだ。

動詞＋「そう」

好像

後ろに乗れそう？

ジュンはまったく起きそうにない。

カーくんはいまにも寝そうだ。

「しか」＋否定文

只是 只有

いま財布（さいふ）の中には10円しか入っていない。

私が信じることができる人はあなたしかいません。

電車（でんしゃ）に乗るしか方法がない。

「だけしか」＋否定文

「だけしか」跟「しか」的意思是一样的。它强调「しか」的意思。

その神社（じんじゃ）にある宝物（たからもの）は、地元（じもと）の住民（じゅうみん）だけしか知らない。

自動詞（じどうし）　と　他動詞（たどうし）

自動詞とは？　什么是《自动词》

它是不用加目的词的动词。

自动词很常被跟助词「は / が」用在一起。

自動詞と他動詞の違い

他动词一定需要目的词和助词「を」或者「に」。

中国語（ちゅうごくご）を話（はな）す　　 他动词

音楽を聞く　　　他动词

音楽が聞こえる　　　自动词

蛇口（じゃぐち）からお湯（ゆ）を出す　　　他动词

家から出る　　　自动词
