日本語の文法の要素

  • 主語（しゅご）　　　　＜例＞「～は」「～が」
  • 述語（じゅつご）　　　　＜例＞「する」「です」「ます」「だ」
  • 名詞（めいし）　　　　＜例＞「わたし」「国（くに）」「家（いえ）」
  • 動詞（どうし）
  • 目的語（もくてきご）　　　　＜例＞「でんわを」「しごとを」
  • 助詞（じょし）　　　　　＜例＞「は」「が」「に」「で」「を」
  • 助動詞（じょどうし）　　　　　＜例＞「ます」「たい」
  • 接続語（せつぞくご）　　　　＜例＞「そして」「だから」「しかし」
  • 副詞（ふくし）　　　　＜例＞「とても」「すこし」「ちょっと」
  • 数詞（すうし）　　　　＜例＞「1個」「1つ」
  • 動詞の活用形（かつようけい）　　　　＜例＞て形　ます形　否定形
  • 原形（げんけい）　　　　　　＜例＞「べんきょうする」「話す」「見る」「開（あ）ける」
  • 過去形（かこけい）　　　　　＜例＞「した」　「はなした」
  • 進行形（しんこうけい）　　　　　＜例＞「している」
  • 受身形（うけみけい）　　　　＜例＞「された」

動詞の活用（どうしのかつよう）
The conjunctions of Japanese verbs

ます形 formal form

point!

The vowel "u" of u動詞 changes into "i"

買う(ka u)　買い(ka i) ます

話す(hana su)　話し(hana shi) ます

聞く(ki ku)　聞き(ki ki) ます

point!

The vowel "e" does not change.

覚える　覚え(obo e) ます

教える　教え(oshi e) ます

開ける　開け(a ke) ます

e形動詞は いくつ ありますか？ How many 'e' verbs below?

します　話します　できます　行きます　来（き）ます　開けます　閉めます　みます　送ります　直します　止まります　止めます　聞きます　読みます

つ

「～します」形で表現（ひょうげん）できる文型（ぶんけい）

  • 希望（きぼう）・願望（がんぼう）の「したい」
  • 「～して・・・」「あれをみて、わたしは～と思いました。」
  • 「～してほしいです」I would like you to V
  • 「～した」「～しました」past tense
  • 「～したことがある」experience
  • 「もし～したら」

＋「て」
「V + て」is used to continue certain subject.

して　話して　できて　行って　来て　開けて　閉めて　みて　送って　直して　止まって　聞いて　聞こえて　読んで　立てて　立って　分かって　別れて　分かれて　飲んで

現在進行形 progresive form 「～ている」

勉強している　話している　言っている

現在進行形 progresive form in polite language 「～ています」

勉強しています　話しています　言っています

＋「ない」　否定形（ひていけい） negative form

怎么用 否定形?（中文） ¿Cómo usar la forma negativa?

しない　話さない　できない　行かない　来（こ）ない　開けない　閉めない　みない　送らない　直さない　止まらない　聞かない　読まない

否定形（ひていけい）の丁寧語（ていねいご）

point!

ます形と同じ形。　ます形の動詞（どうし）＋「ません」


勉強しません　話しません　行きません　来ません（きません）　開けません　ありません　聞きません


「～したい」

point!

ます形と同じ形。

行きたい　見たい　聞きたい　知りたい　勉強したい　買いたい


過去形（かこけい）　「～た」

した（＝「する」　の過去形）　仕事をした　見た　いた　来（き）た

聞いた　覚えた　話した　入（はい）った　

買った　開けた　止めた　入（い）れた

過去形の丁寧語（ていねいご）　polite language in past tense

point!

ます形と同じ形。

聞きました　覚えました　話しました

「～ときに」

point!

動詞の原形＋「ときに」

話すときに　買うときに　出るときに


過去形＋　「～ので」

point!

Don't use the particle「の」 between the verb and 「ので」.

話したので　覚えましたので

経験 ～たことがある

point!

過去形の原形をとる。

行ったことがある　聞いたことがある　泊（と）まったことがある

過去進行形（かこしんこうけい）

point!

「ていた」は「て」形と同じ動詞の活用。

聞いていた　乗っていた

可能形（かのうけい）　「～ができる」「～れる」

point!

u音動詞：　話すことができる　話せる　
　行くことができる　行ける　
読むことができる　読める

言うことができる　言える

eru動詞Ⅰ：　覚える(o bo eru)ことができる　覚えられる

e音動詞：　話せる　書ける　帰れる

聞くことができる　聞ける

意向形（いこうけい）

point!

u動詞の母音がoになる：　行こ(i ko)う　話そ(hana so)う

e動詞は ます形と同じ形。：　覚えよう　伝えよう　開けよう

見よう　来（こ）よう　勉強しよう

意向形（いこうけい）の丁寧語

point!

ます形と同じ形。

行きましょう

覚えましょう

命令形（めいれいけい）　 imperative form

「～しろ」「行け」「読め」像这些形是男生比较用的　「て」形的后面没有词，它就是命令形。
Relatively men use the form like「～しろ」「行け」「読め」.

しろ / して　話して　　行け / 行って　来（こ）い / 来て　開けろ / 開けて　閉めろ　みて / みろ　送（おく）って / 送れ　直（なお）せ / 直して　止まれ / 止まって　聞け / 聞いて　読め / 読んで

＋「て」＋「ください」

してください　話してください　行ってください　来てください

＋「せる」　使役（しえき）

point!

＜u動詞（母音ぼいん　→　a）　＋「せる」＞

まなぶ(bu)　→　まなば(ba)せる　　読む(mu)　→　読ま(ma)せる　　帰る(ru)　→　帰ら(ra)せる

point!

＜e動詞　＋「させる」＞

覚える　→　覚えさせる　　

＋「られる」　受動態（じゅどうたい）

＜u動詞（母音（ぼいん）　→a）　＋「れる」＞

上司（じょうし）から　しかられる

おととい、わたしはパーティーに誘（さそ）われた

＜e動詞　＋「られる」＞

止（と）められる

飲み会（のみかい）に行くことを親（おや）から止められた。

＋名詞　　「～とき」「～こと」「Vたとき」

日本語を話すとき、寝（ね）るとき / ピアノを弾くことができます、アニメをみること / ごはんを食べたとき、カフェに行ったとき

point!

動詞のあとに「の」を入れてはいけない。 Don't put「の」after a verb.

仮定形　　　＋「れば」

自転車に乗（の）れば、つぎの電車（でんしゃ）に 間に合う（まにあう）。

思っていることを ノートに 書けば、気持ちが すっきりするよ。
